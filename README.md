# PoC

Benvenuto nel progetto **PoC**!  
Questa repository contiene una Proof of Concept per il sistema di Magazzini distribuiti.

## Comandi CLI

Spiega qui i comandi CLI disponibili, ad esempio:

| Comando             | Descrizione                         |
|---------------------|-------------------------------------|
| `docker compose up -d --build`         | Avvia il sistema di microservizi     |
| `npm run cli -- list`| Lista l'oggetto specificato con id di un certo inventario            |
